# Sales Data Analysis & Reporting Using Python

## Project Overview
This project analyzes sales data using Python to transform raw sales records
into meaningful business insights for decision-making.

## Tools Used
- Python
- Google Colab
- GitHub

## Tasks Performed
- Created a sales dataset with 50 records
- Calculated total sales amount for each order
- Generated overall revenue and total orders summary
- Analyzed product-wise and category-wise sales performance
- Performed monthly sales analysis using order date
- Exported final dataset as CSV

## Key Insights
- Electronics category generated higher overall revenue
- Certain products showed strong sales performance
- Monthly sales trends help identify business growth patterns
